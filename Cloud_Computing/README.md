\# Suma de Arreglos con OpenMP



Proyecto en C++ que demuestra el uso de programación paralela mediante la librería OpenMP para realizar la suma de dos arreglos.



\## Tecnologías

\- C++

\- OpenMP

\- Visual Studio



\## Descripción

El programa crea dos arreglos, los inicializa con valores y realiza la suma de manera paralela utilizando múltiples hilos de ejecución.



\## Compilación

Es necesario habilitar OpenMP en Visual Studio:

C/C++ → Language → OpenMP Support → Yes (/openmp)
